Rails Composer

A better way to build a Rails starter app.

Download this project as a .zip file Download this project as a tar.gz file

“It's like the Rails generator on steroids.”

Generate an app in minutes using an application template. With all the options you want:

$ rails new myapp -m https://raw.github.com/RailsApps/rails-composer/master/composer.rb

Devise, OmniAuth, CanCan, Twitter Bootstrap, Zurb Foundation, Skeleton, SQLite, PostgreSQL, MySQL, MongoDB, ERB, Haml, Test::Unit, RSpec, Capybara, Cucumber, Turnip, Factory Girl, Machinist, SimpleForm, Gmail, SMTP, SendGrid, Mandrill, Thin, Unicorn, Puma… and more!
